# 1.13.1.Creating_a_file_processing_system
# Ceļazīmju datubāze (db.csv), Konsoles lietotne (README)

Programma ir konsoles lietotne, kas strādā ar failu `db.csv` tekošajā mapē un nodrošina ceļazīmju datu pārvaldību ar izvēlni un lietotāja ievadi. <br>
Tā ļauj: apskatīt faila saturu (izvadot datus tabulas veidā), ierakstīt failā informāciju par jaunu ceļazīmi, izslēgt no faila izvēlēto informāciju un koriģēt (rediģēt) esošas ceļazīmes datus. <br>
Programma atbalsta arī datu sakārtošanu failā pēc datumiem, meklēšanu pēc cenas (atrast visas ceļazīmes, kuru cena nepārsniedz lietotāja uzdoto), kā arī ceļazīmju vidējās cenas aprēķinu.<br>
<br>
Prasības: programmā jāapstrādā lietotāja nepareiza ievade/darbības, pirmkods jāizvieto vienā failā `Main.java` (bez `package`), un faila sākumā jābūt komentāram ar autoru (apliecības numurs, vārds, uzvārds, grupa). <br>
Programma apstrādā tikai `db.csv` failu un neveido grafiskus logus.<br>
